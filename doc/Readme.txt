=================================
        FUBI documentation
(Full Body Interaction Framework)
=================================

To generate the API documentation, please install:
 - Doxygen: http://www.doxygen.org
 - Graphviz (optional): http://www.graphviz.org
Add their bin directories to your path and execute "GenerateAPIDoc.bat" (or "GenerateAPIDoc.sh" on linux)

To generate the XML documentation, please install:
 - XSDDoc: http://xframe.sourceforge.net/xsddoc
Add its bin directory to your path and execute "GenerateXMLDoc.bat" (or "GenerateXMLDoc.sh" on linux)
